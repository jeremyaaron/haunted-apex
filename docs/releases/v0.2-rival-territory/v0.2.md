# Haunted Apex v0.2.0 — Rival Territory

## Iteration thesis

> **The same action should feel different depending on where you do it and whose territory you touch.**

v0 proved the core pressure loop.
v0.2 should make the game feel less like abstract meters and more like a city with contested spaces.

The point is not to add lots of lore. The point is to add **targets**.

Current v0:

```text
Action + Operative → Outcome
```

v0.2:

```text
Action + Operative + Target → Outcome + Rival Reaction
```

That is the next meaningful layer.

# Target modifier rules

Keep the modifier system simple and deterministic.

## Venue modifiers

```ts
function applyVenueModifiers(effects: PressureDelta, venue: Venue): PressureDelta {
  return {
    ...effects,
    resources: effects.resources ? effects.resources + venue.wealthMod * 150 : effects.resources,
    intel: effects.intel ? effects.intel + venue.intelMod : effects.intel,
    heat: effects.heat ? effects.heat + venue.heatMod : effects.heat,
    loyalty: effects.loyalty ? effects.loyalty + venue.loyaltyMod : effects.loyalty,
    ruin: effects.ruin ? effects.ruin + venue.ruinMod : effects.ruin,
  };
}
```

## District modifiers

Districts should be subtler than venues.

```ts
function applyDistrictModifiers(
  effects: PressureDelta,
  action: ActionDefinition,
  district: District,
): PressureDelta {
  const next = { ...effects };

  if (effects.resources) {
    next.resources = effects.resources + Math.round((district.wealth - 50) * 8);
  }

  if (effects.intel) {
    next.intel = effects.intel + Math.round((district.secrecy - 50) / 15);
  }

  if (effects.heat && effects.heat > 0) {
    next.heat = effects.heat + Math.round(district.heat / 20);
  }

  return next;
}
```

This makes districts matter without making them impossible to read.

# Rival attention

If the target is controlled by a rival, that rival gains pressure.

```ts
function getTargetController(state: GameState, target?: ActionTarget): Rival | undefined {
  if (!target) return undefined;

  if (target.type === 'venue') {
    const venue = getVenue(state, target.id);
    return venue.controllingRivalId ? getRival(state, venue.controllingRivalId) : undefined;
  }

  if (target.type === 'district') {
    const district = getDistrict(state, target.id);
    return district.rivalId ? getRival(state, district.rivalId) : undefined;
  }

  if (target.type === 'rival') {
    return getRival(state, target.id);
  }
}
```

Pressure gain:

```ts
function calculateRivalPressureGain(actionId: ActionId, targetType?: string): number {
  const baseByAction: Record<ActionId, number> = {
    gather_intel: 4,
    run_small_job: 8,
    bribe_official: 5,
    recruit_operative: 3,
    expand_influence: 12,
    lay_low: 0,
  };

  return baseByAction[actionId] ?? 0;
}
```

Player-facing labels:

```ts
function rivalPressureLabel(pressure: number): string {
  if (pressure < 25) return 'Watching';
  if (pressure < 50) return 'Interested';
  if (pressure < 75) return 'Provoked';
  return 'Retaliating';
}
```

# Agent/simulation updates

Your trial agents should now have target-selection logic.

## Existing agent types

Preserve:

```text
Sane
Cautious
Aggressive
Greedy
Random
```

## Add target preferences

### Sane agent

```text
Chooses targets with best reward/risk ratio.
Avoids rival-controlled targets when rival pressure >= 60.
Prioritizes low-heat targets if Heat >= 65.
```

### Cautious agent

```text
Prefers Pale Circuit and low-heat districts.
Avoids rival-controlled targets.
Likely fails Dominion target.
```

### Aggressive agent

```text
Targets highest Dominion reward.
Does not care about rival pressure until Heat >= 85.
```

### Greedy agent

```text
Targets highest Resource reward.
Favors Zero Mercy and Black Halo Exchange.
```

### Random agent

```text
Random valid action + random valid target.
```

# Balance expectations for v0.2

The v0.2 tuning target should be similar to v0, but slightly more volatile.

Expected:

```text
Sane strategy: 55–75% win rate
Cautious strategy: low win rate due to Dominion shortfall
Aggressive strategy: 35–60% win rate, high Heat losses
Greedy strategy: 30–55% win rate, swingy
Random strategy: 5–12% win rate
```

If sane strategy drops below 45%, rivals/targets are too punitive.

If aggressive wins above 75%, rival pressure and Heat are too soft.

If cautious wins often, Dominion target is too easy or safe targets are too rewarding.

If target choice barely changes outcomes, venue modifiers are too weak.
